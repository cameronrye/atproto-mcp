üí° Design Concept

The logo combines several key elements that represent the project's purpose:

  • Hexagonal Shape: Represents structure, connectivity, and the decentralized nature of the AT Protocol
  • @ Symbol: Central element representing the AT Protocol and its identity-focused architecture
  • Network Nodes: Small circles connected to the center represent the Model Context Protocol's role in connecting LLMs
  • Blue Gradient: Uses Bluesky's brand colors to maintain visual consistency with the AT Protocol ecosystem
  • Modern & Clean: Professional design suitable for technical documentation and developer tools
